Show/Hide your WordPress block content by various conditions.

== Description ==

Block conditions provide you various conditions for show/hide your block. Like hide in specific countries, For logged in user or logged out user, For specific devices Like mobile,desktop,tablet. It gives you  an amazing experience to edit posts and restrict content for device,countries,user state specific.

Note: This plugin use the <a href="http://ip-api.com" rel="block-conditions" title="IP-API">http://ip-api.com</a> website to fetch user's IP address and Location and <a href="https://api.ipify.org/" rel="block-conditions" title="API-IPIFY">https://api.ipify.org/</a> for IP Address.

## What can Block conditions do?

Included Conditions:

* Show or Hide block based on the user logged in or logged out.
* Show/Hide block in mobile/desktop/tablet.
* Hide block in specific countries.

== Installation ==

Block Conditions Installation Instructions:

1. Upload the Block Conditions plugin to your /wp-content/plugin/ directory or through the plugin admin section under "add new".,
2. Activate the plugin through the ‘Plugins’ menu in WordPress.,

== Frequently Asked Questions ==

= Which block i can hide conditional? =

Almost all blocks you can show/hide  conditionally even if they are Third party plugin or Default WordPress blocks.
